# sankhya-helper

Utilitário para formatar objetos JavaScript no padrão de JSON da [API de integrações Sankhya](https://developer.sankhya.com.br/reference/api-de-integra%C3%A7%C3%B5es-sankhya).

# Instalação

```
npm i sankhya-helper
```
